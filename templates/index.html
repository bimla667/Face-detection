Why This Solution Matters

Built for the real challenges of India's highway workforce

🛣️

Remote Locations

Highway toll booths, construction sites, and maintenance depots often lack reliable internet. This system works completely offline.

🚫

Prevents Proxy Attendance

Liveness detection (blink, head movement, smile) ensures the person present is physically real — not a photo or video.

⚡

Instant Verification

Sub-second face recognition using dlib's 128-dimension face encoding model — fast enough for high-traffic checkpoints.

🔒

Data Privacy

All biometric processing happens locally. Face encodings never leave the device unless explicitly synced to an authorised cloud.

📱

Low-Cost Hardware

Runs on a Raspberry Pi 4 or basic laptop. No GPU required. Total hardware cost under ₹5,000 per unit.

☁️

Smart Sync

Attendance records queue locally and automatically upload to AWS when connectivity resumes — zero data loss.
